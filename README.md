🛠 Mastering API Testing with Postman
Beyond the code, this project was built to deepen my expertise in Postman as a core part of the development lifecycle. I utilized Postman to ensure the reliability and security of every endpoint.

Postman Implementation Details:
Environment Variables: I configured dynamic environments (Local, Staging) to manage base URLs and sensitive keys without hardcoding values.

Automated Testing: Wrote Postman scripts in JavaScript to automatically verify status codes, response times, and data structures after every request.

Authentication Flow: Mastered the use of Collection Variables to automatically capture Bearer tokens from login responses and inject them into subsequent requests.

Collection Documentation: Organized requests into folders with detailed descriptions, providing a "living document" that reflects the API's current state.
